Una caja tiene 4 partes:

Contenido

Dentro de este, se le puede asignar un padding, Relleno o acolchamiento alrededor de este.

Alrededor del padding se situa el Borde

Y justo despues el Margen

Como ayuda para controlar el tamaño de las cajas se creó en CSS3 el box-sizing.
Las cuales se pueden controlar con: content-box | padding-box | border-box.

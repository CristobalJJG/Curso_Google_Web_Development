Tablas

Etiquetas más utilizadas:

Se utiliza "table" como creación de la tabla.

Se utiliza "tr" (Table Row) como fila de la tabla.

Se utiliza "td" (Table Data cell) como Celda de datos.

Extras:

Para añadir un Header se puede usar "th" (Table Header cell) Celda de cabecera.

"thead" agrupa una o más filas de cabecera.

"tbody" agrupa una o más filas de cuerpo.

"tfoot" agrupa una o más filas de final.

"caption" añade un título a la tabla

"colspan" y "rowspan" se utiliza como "combinar celdas".
